Terminal Based Chat App
This project began when I wanted to have a chat server built on Python. However all the tutorials I saw required port forwarding something that I did not have that option to use. I thus made a peer-to-peer based Python Chat Client that works without the need for complex router settings.

TeachablePy

A simple AI chatbot

I wanted to build a chatbot that was better than a hardcoded bot, but easier to build than a bot based around a Neural Network or similar. This project was the result. This program can be taught how to respond to questions and it stores all answers in a csv file allowing sharing of custom chatbots.
